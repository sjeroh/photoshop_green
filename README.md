# 포토샵 기능 메모

```
깃허브 안에 네모 만들기는 탭위의 `~키 3번 누르면 됨 그 안에 써넣기 - 옆에 복사 가능한 키도 뜨게 됨
alt + 마우스 휠 조정 -> 화면 크기 조절
```


* 별표 치고 띄우면 앞에 점 생기면서 메모도 가능
- 대쉬로 치고 해도 가능

기본적으로 그냥 쳐도 메모 가능
